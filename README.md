## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Ar9an/arpan-portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```
